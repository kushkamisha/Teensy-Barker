# Teensy-Barker parser

## About
Collector of menu files from restauraunt's websites

## Start

`$ npm start`

## Documentation

To generate up-to-date documentation run
`$ npm run getdocs`

And then open it with
`$ npm run readdocs`

## Test
`$ npm test`
